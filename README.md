# Random 1:1 Video Chat MVP (WebRTC + Socket.io)

## Run (Windows/macOS/Linux)
1) Install Node.js 18+
2) In this folder:
   - npm install
   - node server.js
3) Open:
   - http://localhost:3000

## Notes
- Works on localhost without HTTPS.
- For real deployment, HTTPS is required for camera/mic permissions.
- For production reliability, you likely need a TURN server.
